    Skill Sequence



    • Skill sequence system has been added.
      • Skill sequence UI can be opened by clicking the Sequence button in the Skill UI.
      • Skill sequence can be used to cast many different skills quickly and consecutively with one button.
      • Multiple skill sequence preset can be made for different situations.
      • Registered sequences can be viewed by pressing the exclusive skill sequence key and the skill sequence hotkey at the same time.
      • In area where skill usage is restricted, skill sequence will be restricted as well.
    • How to use skill sequence:
      1. Click the sequence button at the bottom of the Skill UI.
      2. Click the layout setting at top right and set the number of rows.
      3. Click a sequence and register desired skills.
      4. If a hotkey is set for the sequence, the skills will cast from left to right order.

    • Skill sequence rules:
      • Once the sequence key in inputted, the character will do an exclusive animation and cast all registered spells with a set interval between each skill.
      • All cast animation effects will be displayed simultaneously.
      • Skills that are not executed will not have their effects displayed.
      • If a skill in a sequence is on cooldown, casting the skill sequence again will not activate that skill.
      • Skills with a recast component cannot be added into Skill Sequence.
      • You can use the sequence as long as at least one skill in the sequence is off cooldown.
      • When using the sequence, only usable skills will be activated.
      • If there are unusable skills due to lack of Resource/MP, the remaining skills will still be cast.
      • Display system message "[skill name] is not usable” .
      • If the skill sequence includes skills that are not usable in the air, activating the skill sequence in the air will not activate that skill.
      • If the sequence has already activated, all skills will be activated regardless of status effects.
      • After activating the sequence, skill activation is no longer affected by status effects.
      • After using the sequence, being hit by a status effect does not cancel skill activation.
      • Skills that have interaction with arrow keys being pressed cannot use this functionality when cast inside a skill sequence.
      • When registering a skill with custom command functionality inside the skill sequence, it is used according to the command on/off status.

    某一天家裡突然多出了一盒上面有周杰輪、菜依琳、黃綠紅跟孫燕珠的「新星勢力包」，開啟了我跟這個蘑菇屋圖示的遊戲的邂逅

    󠀠

    印象中那時候跟幼稚園班上的同學聊天，不知道要選哪個伺服器，被推薦去了綠水靈，開了一隻弓箭手「小仁豪4」，好不容易練到10等，結果被我姊告知伺服器選錯，她在蝴蝶精……，只好又重創一隻「小仁豪7」，骰了骰子、選了法師。

    󠀠

    按照密笈教的，從維多利亞三叉路的草叢走進隱藏地圖，結果在肥肥海岸打怪的時候發現緞帶肥肥怎麼打都打不動，於是能力值全點了力量，硬是玩到了16、7等才發現鑄成滔天大錯XDD

    󠀠

    也因為那時候家裡電腦只有一台、也被家裡限制玩電腦的時間，最後也就無疾而終。（結果長大之後跑回去蝴蝶精玩，覺得小仁豪7名字太恥直接把角色給砍了，而小仁豪4逃過一劫）

    󠀠

    後來則是看著姊姊開始玩刀賊，好不容易玩到30等，買了GASH點卡打扮自己的角色，結果過幾天發現自己全身脫光光在自由市場

    姊弟倆一起在電腦前震驚，向司馬遷領了免費救急租用裝備之後我姊直接崩潰不玩了XD

    󠀠

    又過了一年多(2008年新年)，我姊又重開了一隻法師準備練僧侶，整個寒假都在練等，火獨眼獸刷刷刷、超綠刷刷刷、怪物擂台刷刷刷，春節刷餃子、情人節刷巧克力、英語村刷字母、周年慶打蛋糕跟楓葉，翻著密笈刷天空之城塔、愛奧斯塔的卷軸，每天去上海灘交雞爪跟鴨蛋。小時候沒感覺，長大才發覺我姊那時候是真的很有毅力XDD

    󠀠

    過了一陣子家裡電腦中毒，有好一大段時間沒得玩，再回去已經是BB大改機甲戰神的事了。

    󠀠

    變成新楓之谷之後，只覺得地圖變好多、介面變好酷，練等變好好練，所以又滿心歡喜地跑回去玩，雖然每次都玩得不精也玩得不久，但也成了每年寒暑假的定番之一。

    (印象中那陣子佛到有一種放到現在來看都覺得是不是快倒的感覺，大改前都會有甚麼回歸活動，一段時間沒玩的帳號都可以拿獎勵，記得好像拿到過1000楓點)

    󠀠

    後來還有薑餅人泱泱、紺紫小惡魔之類的永久寵。

    直到看到朋友衝星力、衝卷軸、洗方塊之後才意識到遊戲生態變得不一樣了，不過因為每次回谷連燃燒130等都練不完，所以也沒什麼概念。

    󠀠

    再一次回來，已經是大學畢業後的事情了，我姊又跑回來玩，問我要不要一起。

    那時候剛好是冒險家D改後沒多久，我選了一個從來沒玩過的聖騎士，本來想說這次大概也是玩得有一陣沒一陣的之後就會離開，結果看到了韓版NEW AGE發佈會的直播，看到自己從小玩到大的遊戲還能有新的變化很感動也很驚喜，決定一頭栽下去大玩特玩，實體活動也是能參加就參加(楓潮音樂節、楓潮派對等等的)，就一路到了現在。

    󠀠

    回來兩年多，默默的把本尊練到285、創世解放，戰地也9000了。

    從打培羅德只有1、打任何王都會一直死出去，到現在可以不用天上打完困8，打到一件新裝備、慢慢地用活動獎勵洗出來再穿上去的成就感，有許許多多的喜悅都是楓之谷帶給我的。

    󠀠

    看著自己的遊戲帳號也滿19年了，衷心祝福也期待楓之谷繼續陪我們下一個20年！祝楓之谷20歲生日快樂！
