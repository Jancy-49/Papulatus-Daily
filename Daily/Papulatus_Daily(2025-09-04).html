※ 참고 사항

* 우수 신고자는 매달 1회 운영정책 위반 단속 결과 공지에 함께 안내됩니다.

* 선발 기준 및 유효 신고 횟수 등에 따라 우수 신고자 선발 인원은 매달 달라질 수 있습니다.

* 넥슨캐시 보상은 9/4(목) 오후 7시 이후 순차 지급될 예정입니다.

* 메이플포인트는 위 기재된 메이플ID로 지급되었으며, 게임 아이템은 유효 신고건을 접수한 캐릭터 월드의 메이플 보관함으로 지급되었습니다.

(보상 아이템이 지급되기 이전 캐릭터 삭제, 캐릭터명 변경을 진행한 경우 최근 접속한 캐릭터 월드의 메이플 보관함으로 지급됩니다.)

* 게임 아이템의 보관함 보관 기간 및 유효기간은 2025년 10월 4일(토) 오후 11시 59분까지로 꼭 기간 내 수령 부탁드립니다.

* 내가 바로 메이플 캅스 훈장 교환권 사용 시, 획득하는 아이템은 교환 불가, 영구입니다. 꼭 원하는 캐릭터로 교환권을 사용해주세요.

* 동일 명의 내 여러 메이플ID가 보상 지급 대상자로 선정된 경우, 가장 상위 보상 1개만 지급됩니다.

* 유효 신고 항목은 ‘욕설/음란글’, ‘광고’, ‘현금거래’, ‘불법프로그램사용 제보’에 한해서만 집계됩니다.

* 유효 신고 횟수는 메이플ID 기준으로 집계되며, 신고 이후 일정 기간 이내에 제재된 경우에만 유효 건으로 인정됩니다.

* 신고를 진행한 달 내에 제재까지 완료된 케이스만 유효 신고 건 수로 집계됩니다.

* 허위 신고, 무분별한 신고 등 비정상적인 신고 내역이 확인될 경우 선정 대상에서 제외될 수 있습니다.

* 보상 지급 이전 넥슨ID 또는 메이플ID를 탈퇴한 경우 보상이 지급되지 않습니다.

 

앞으로도 메이플스토리에서는 현금거래 행위, 광고/홍보성 채팅 등 

운영정책에 위반되는 행위를 근절하기 위해 꾸준히 단속을 진행하겠습니다.


운영정책에 위반되는 행위를 확인하셨다면 [신고하기] 기능을 통해 적극적인 신고 부탁드립니다.

 

- 신고 방법 : 운영정책 위반 대상자 캐릭터 우클릭 -> 신고하기



 

항상 쾌적하고 깨끗한 메이플스토리를 위해 노력해주시는 勇士们께 감사드립니다.

[什麼是人體模型？]

1) 人體模型是儲存角色目前的髮型/臉型/皮膚後，隨時能讀取的系統。

2) 購買人體模型時，可在[選單] → [美容商店] → [人體模型]中使用。

3) 購買人體模型時，該角色在沒有期間/次數限制的情況下，可持續使用儲存的髮型/臉型/皮膚。

4) 髮型的顏色不同時，可保管相同的髮型。整形時，隱形眼鏡顏色不同的話，可保管相同的臉部風格。皮膚的部分，皮膚的顏色不同時，可保管相同的皮膚。

5) 套用保管的髮型/臉型/皮膚時，目前角色配備的髮型/臉型/皮膚道具，以及人體模型保管的道具會互相交換。

6) 該組合包每個帳號可購買3次，每個角色最多可開啟23個人體模型欄位。

7) 人體模型只能套用在購買的角色，相同帳號的角色不會共享。

[NEW] MapleStory Universe Open API Release (v1rc1)


 

We are pleased to announce the release of the latest version of the MapleStory Universe Open API (v1rc1).

 

This update includes expansions to existing APIs as well as the introduction of new API categories, broadening the ways builders can create, analyze, and enhance experiences in the MSU ecosystem.

 

 

■ Key Updates

Compared to the previous v1beta, the new v1rc1 version includes:

  • 28 endpoints in total (previously 8) 
  • Expanded data visibility: Retrieve detailed wallet data, character stats, in-game metadata, and token-level item information including enhancement history and ownership.
  • Access static game data (skills, quests, item sets, categories) through GameMeta API endpoints. 

 

 

■ API Keys

Builders can now request API keys by applying here: [Link]

Keys will be issued starting on September 11, 2025 and distributed sequentially. Please note that depending on application volume, issuance may take longer.

 

 

■ Where to Learn More

For full endpoint details and usage examples, please refer to the official documentation:
👉 MapleStory Universe Open API Documentation

 

 

Thank you to all the builders who shared their insights and helped guide these updates.

These additions are the result of valuable feedback and requests from our builder community.

We deeply appreciate your collaboration and remain committed to continuously improving the tools you rely on.

 

Your contributions continue to shape the MapleStory Universe into a more open and collaborative ecosystem. 

We look forward to seeing the new experiences you create with these APIs!

 

 

⚠️ Important Notice

※ The current v1beta API will no longer be supported after December 31, 2025.

※ As of September 11, 2025 (upon v1rc1 key issuance begins), all new builder applicants will be issued keys for v1rc1 by default.

※ Some response formats have changed in v1rc1 compared to v1beta. We strongly encourage all existing builders to reapply for a new API key and migrate to v1rc1 to avoid future service disruptions.

※ The v1rc1 schema may be subject to further updates. Additional endpoints may also be introduced to support new features and use cases. We recommend builders regularly check official announcement for any changes or additions.
