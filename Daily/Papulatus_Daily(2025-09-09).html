    Known Issues Following the September 3 Patch


     

    Here is an update on known issues on MapleStory N. Once the issue is resolved, you will be informed by adding the time of issue resolution in front of the according issue.

     

     

    In Progress

    • An issue where the game may experience problems if it is closed quickly without selecting a Potential or Bonus Stats after using a Black Cube or Black Rebirth Flame to enhance equipment.
      1) Potential or Bonus Stats is not properly applied in the game.
      2) The equipment cannot be equipped, unequipped, moved, or enhanced.
      • When using a Black Cube or Black Rebirth Flame to enhance equipment, please exit the game 1–2 minutes after selecting a Potential or Bonus Stats.
      • This issue is scheduled to be fixed in an upcoming maintenance. In the meantime, separate processing will be carried out sequentially. 
        For Maplers who wish to resolve this issue more quickly, please contact Official Support, and we will provide detailed information and assistance.

     

     

    Resolved

    • An issue where an additional “Fiesta Coin received” message is shown when claiming the Day 2, 4, 6, 8, and 10 rewards
      • Please note that the message may indicate that you've received Fiesta Coins, but only the appropriate reward items for that day will be distributed without the Fiesta Coins.

    [UPDATED August 21, 2025 03:00(UTC+0)] Known Issues Following the July 23 Patch


     

    Here is an update on known issues on MapleStory N. Once the issue is resolved, you will be informed by adding the time of issue resolution in front of the according issue.

     

     

    Resolved

     

    • An issue where the 'Heroes Fiesta - Weekly Mission' is not marked as completed despite meeting the completion conditions has been occurring for some characters since 00:00 (UTC+0) on July 31.
      • [UDATED August 1, 2025 02:40] This issue is now resolved.
      • For Maplers who have already encountered the issue, the Weekly Mission can still be completed by other characters in the account that has Weekly Mission in [In Progress] state.
      • [UDATED August 4, 2025 06:50] Using the Gift Box will automatically mark completion on the Weekly Missions that should have been completed previously.
        • Detailed information of the reward distribution will be provided via a separate notice after the distribution is complete.

     

    • An issue where the 'Repair Busted Surveillance Tower' daily quest in Haven cannot be progressed.
      • Quest with the Issue:
        • Scrapyard: Repair
        • Skyline: Repair
        • Black Heaven Inside: Repair
      • Please proceed with the Kritias or Dark World Tree daily quests until the issue is resolved.

     

    • An issue where entering Legion Raid was prevented for some Maplers with an error popup stating that rank rewards should be received prior to entering Legion Raid.
      • [UPDATED July 28, 07:35] The issue with the entry has been resolved. Please try entering Legion Raid again.

     

    • An issue where the Skill Points for Sniper and Ranger are insufficient to allocate for Reckless Hunt skill.
      • For these jobs, there will be a Skill Point reset upon reconnecting to the game to allow allocation for the skill. Further announcement will be provided once the fix has been applied.
      • [UPDATED July 24, 2025 07:24(UTC+0)] Skill Points will reset upon reconnecting to the game.

     

    • An issue where Maple Legion-related quests cannot be accepted in Fang world
      • There is currently an issue where Maple Legion-related quests cannot be accepted in Fang world.
      • For Maplers in Fang world, there is a resolution in progress that will allow accepting Maple Legion-related quests via Maple Administrator. Separate announcement will be provided regarding the resolution once the test is complete.
      • [UPDATED July 24, 2025 07:24(UTC+0)] As a temporary measure, you may visit Maple Administrator in Henesys to accept and proceed with Maple Legion-related quests.
      • [UPDATED July 25, 2025 01:00(UTC+0)] The Union Quest menu has been removed from the Maple Administrator following the July 24 temporary maintenance

     

    • An issue where an excessive number of monsters were spawning in certain maps in Haven due to a problem in the personalized hunting fields system.
      • This issue is scheduled to be resolved during the temporary maintenance on July 24.
      • [UPDATED July 25, 2025 01:00(UTC+0)] Some monsters taht had connected spawn animations with other mosters will now spawn without these animations.

     

    • An issue where after fixing Thunder Sphere in place, Thunder Sphere cannot be summoned again in a fixed place until it completely disappears.
      • In order to summon Thunder Sphere in a fixed place, you need to either right-click the Thunder Sphere icon at the top right of the game screen or wait until the Thunder Sphere duration ends and it disappears.

     

    • An issue where certain coupon items purchased from the Fiesta Coin Shop fail to register on the Raffle Dashboard upon use.
      • Coupons with the Issue:
        • Gear UP: Reloading Basic Raffle Coupon
        • Gear UP: Reloading Premium Raffle Coupon
        • Fiesta Basic Pet Raffle A Coupon
        • Fiesta Basic Pet Raffle B Coupon
        • Fiesta Premium Pet Raffle A Coupon
        • Fiesta Premium Pet Raffle B Coupon
      • Coupons purchased from the Fiesta Coin Shop are currently unable to use. Further guidance will be provided once they become available for use again.
      • For Maplers who used purchased coupons and failed to register raffles, the consumed coupons will be reissued after the issue has been resolved.
      • [UPDATED July 25, 2025 01:00(UTC+0)] The reissued schedule will be announced at a later date.
      • [UPDATED July 25, 2025 05:15(UTC+0)] The coupons purchased from the Fiesta Coin Shop that were consumed but failed to register for the raffle have been reissued.
      • [UPDATED July 25, 2025 05:15(UTC+0)] Please make sure to claim the reissued item before the storage period ends: July 29, 23:59 (UTC+0).

     

    • An issue with discrepancy between the in-game skill description and the July 23 Patch Note for the skill: Dark Thirst.
      • The values specified in the July 23 Patch Note are currently applied for Dark Thirst.

     

    • An issue where the game client would close when using Fiesta Event enhancement items purchased from the Fiesta Coin Shop while the language was set to Thai.
      • This issue is scheduled to be fixed through the temporary maintenance on July 24.
      • Items can be used normally when the game is set to languages other than Thai.

     

    • An issue where the game client crashes upon rebinding a large quantity of NFT items in the NFT Collection.
      • When rebinding a large quantity of NFT items, please uncheck all target items first and then select less number of items in total for rebinding.
