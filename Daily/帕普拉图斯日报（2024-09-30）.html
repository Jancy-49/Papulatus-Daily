你好. 这里是冒险岛

 

2024년 11월 5일(화)부터 

메이플ID 로그인 서비스 이용에 변경점이 있어 사전 안내해 드립니다. 

 

그간 넥슨 이메일ID로 전환하지 않은 기존 넥슨ID를 이용하고 있는 용사님께서는 

넥슨ID에 연동된 각 메이플ID로 로그인해야 접속할 수 있어 

넥슨ID 로그인 정보와는 별개로 메이플ID 로그인 정보를 별도 관리해야 하는 불편함이 있었는데요. 

 

위와 같은 로그인 과정의 불편함을 개선하기 위해 

넥슨ID로 로그인 후, 넥슨ID에 연동된 메이플ID를 선택하여 

게임에 접속할 수 있도록 변경될 예정입니다. 

 

- 대상자: 메이플ID 로그인을 이용하고 있는 용사님 (* 이메일 형식이 아닌 넥슨ID 이용 대상자) 

- 변경사항: 메이플스토리 홈페이지, 게임런처, 메이플핸즈+, 메이플스토리M 연동 로그인 방식 변경 

변경 전 

넥슨ID에 연동되어 있는 메이플ID로 로그인 

변경 후 

넥슨ID로 바로 로그인 

※ 로그인 방식 변경 후에도 기존 메이플ID 내 게임 정보는 그대로 유지됩니다. 

※ 넥슨 이메일ID, 외부계정(구글/페이스북/네이버/애플) 로그인에는 변경사항이 없습니다. 

※ 로그인 방식 변경 후에도 메이플ID 중복 로그인은 가능하지 않습니다. 

※ 로그인 방식 변경 후 이메일 형식이 아닌 넥슨ID로도 넥슨플러그 로그인 후 게임 실행이 가능합니다. 

 

 

메이플ID에 연동되어 있는 넥슨ID와 비밀번호를 알고 있다면 

변경된 로그인 방식으로 접속 후 기존 메이플ID 게임 정보 그대로 이용이 가능합니다. 

만약 메이플ID에 연동되어 있는 넥슨ID와 비밀번호가 기억나지 않으시더라도

リブートワールドに関する大切なお知らせ


日ごろは『メイプルストーリー』をご愛顧いただきありがとうございます。

リブートワールドの今後の運営方向について重要な変更があり、お知らせします。
メイプルストーリーは今年サービス21周年を迎え、多くのメイプラーに喜びを与えることを目標に、
継続して新規/上位コンテンツを提供するよう努めています。
その反面、新しいコンテンツが追加されるほど、一般ワールドとリブートワールドとの特性の違いにより、
すべてのメイプラーに良質なサービスを提供するのが難しいと判断しました。

そのため、リブートワールドを一般ワールドと同じ環境になるよう各種仕様の変更を実施いたします。
この対応の方針をお知らせすることになり、リブートワールドのメイプラーの皆様には大変申し訳なく思います。

今回の変更事項は現在リブートワールドを利用しているメイプラーにとって非常に大きな変化であるだけに、
どのような苦悩と意図をもとに変更作業を行う予定か事前にご案内いたします。
リブートワールドが一般ワールドと同じ環境に変更されるパッチは、10月、12月、1月の計3回にわたって進行する予定であり、
それぞれのパッチの方向性と意図について説明させていただきます。

・リブートワールド1次アップデート（2024年10月30日）
　-1次アップデートは、リブートワールドを一般のワールドに切り替える前に、

拉仇恨的、输出的站在相反方向。吸引仇恨的人在“时间操纵术”发动的过程中引导半半，站在黑色时钟位置，为团队争取到更多的挑战时间。

 

 

第二阶段

首先，记一个时钟刷新的小诀窍。

时钟出现顺序依次为15243，提前站到下一个点位，就能最快地争取到最多时间，最高能增加5分钟。

当时钟都是白色，就需要停止攻击了，注意取消放置类的技能释放。

 

其次，熟悉下半半的攻击机制，A三下丢技能，进入下一轮循环。

第一轮飞扇翅膀。

第二轮平A三下拉时钟然后飞。

第三轮平A三下，飞天捶地眩晕。

这样说大家能更好地对照理解。

 

 在半半第三阶段施展“内心世界”时，可在半半人物名称词条——“混”字三点水的位置进入，但是里面打不完出来小队都会遭殃，所以只要保证输出够秒杀内心世界的半半即可。

通过内心世界，到五分之一左右血量进入“原子弹”掉落阶段，“原子弹”会对我们造成极高的伤害，注意躲避就能轻松拿捏半半了！

 

最后，要提醒大家注意生存！如果队伍没有光之魔导师，需备好药水，防止力竭。有光魔时，光魔应在关键时刻放弃攻击，专注辅助，解除半半的眩晕效果，确保队友的存活和输出时间。
